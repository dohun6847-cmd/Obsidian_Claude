# 5. Zettelkasten — 운영 가이드

## 목적

**지식 발전소**입니다. 외부 자료를 받아들이고, 소화하고, 나만의 영구 지식으로 정제하는 3단계 파이프라인입니다.

ZK는 PARA(1~4번 폴더)와 명확히 분리됩니다:
- **PARA**: 시간 기반 실행 작업
- **ZK**: 시간 독립적 영구 지식 축적

## 3단계 파이프라인

```
외부 세계
    │
    ▼
[00. Inbox]  ← 빠른 캡처, 가공되지 않은 생각
    │
    │ 정리 (주 1회)
    ▼
[10. Literature]  ← 원천 자료 기반 요약 (저자의 생각)
    │
    │ 해석 & 연결
    ▼
[20. Permanent]  ← 나만의 통찰 (내 언어, 내 관점)
```

## 핵심 원칙

| 원칙 | 설명 |
|------|------|
| 원자성 | 하나의 Permanent Note = 하나의 아이디어 |
| 연결 | 새 Permanent Note는 최소 2개 기존 노트와 연결 |
| 고유성 | 나만의 언어로, 나만의 해석으로 |
| 시간 독립성 | 프로젝트가 끝나도 지식은 남는다 |

## ZK ↔ PARA 흐름

- **PARA → ZK**: 프로젝트/영역 작업 중 얻은 통찰을 Permanent Note로 정제
- **ZK → PARA**: Permanent Note를 `[[링크]]`로 참조해 프로젝트 결과물 작성

**복사 금지** — 항상 `[[링크]]` 또는 `![[임베드]]`로 참조합니다.

## 하위 폴더

- **[[00. Inbox/CLAUDE|00. Inbox]]** — 빠른 메모, 임시 캡처
- **[[10. Literature/CLAUDE|10. Literature]]** — 원천 자료 노트
- **[[20. Permanent/CLAUDE|20. Permanent]]** — 영구 지식 노트
